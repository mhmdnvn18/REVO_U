# Membuat WEB dengan HTML
  1. Buka aplikasi Visual Studio Code
  2. lalu click Create New file > lalu pilih HTML
  3. setelah itu ketikan kode seperti di bawah ini :
     <!DOCTYPE html>
     <html>
        <head>
          <title>LAUNDRY</title>
        </head>
        <body>
          <h1> Hello World! </h1>
          <h3> Are you happy? </h3>
       </body>
      </html>
